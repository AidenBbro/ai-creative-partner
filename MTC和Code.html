什么是 Trae Solo？

Trae Solo 是字节跳动推出的AI原生IDE产品，提供Code和MTC两种核心模式，通过更轻量、更直观的交互范式，为用户提供沉浸式且高效的AI驱动开发体验。

核心定位：覆盖专业开发到日常办公的全场景AI助手

两种核心模式

MTC 模式

More Than Coding

面向非开发岗位用户，处理文档、数据、演示稿等泛开发场景。适合产品经理、数据分析师、运营人员等。

Code 模式

Professional Coding

面向专业开发者，继承SOLO Agent核心能力，自主分析项目架构、编写复杂代码、调试纠错等。

核心对比

对比维度 MTC 模式 Code 模式
定位 办公/文档助手 程序员助手
目标用户 产品经理、数据分析师、运营等 专业开发者、程序员
权限 沙箱环境（仅授权文件夹） 本地全权限（可运行命令）
核心能力 文档、PPT、数据、方案 代码、调试、架构、部署
终端支持 桌面端 + 网页端 + 手机端 桌面端 + 网页端
交互方式 动嘴提需求，AI执行 Agent主导，可深度定制

适用人群

🎯 MTC 模式适用

  • 📋 产品经理 - 需求文档、PRD、用户故事
  • 📈 数据分析师 - 数据处理、可视化、报告
  • 📣 运营人员 - 活动方案、调研报告、复盘
  • 🎨 设计师 - 设计说明、提案文档
  • 📚 行政人员 - 工作报告、会议纪要

💻 Code 模式适用

  • 🔧 前端开发者 - React/Vue/Angular项目
  • ⚙️ 后端开发者 - API、数据库、服务端
  • 🧪 全栈工程师 - 端到端开发
  • 🔍 测试工程师 - 自动化测试、调试
  • 🚀 DevOps工程师 - 部署、CI/CD

如何选择？

选择 MTC 模式

  • 不需要写代码的工作
  • 需要处理大量文档和数据
  • 想要快速生成汇报材料
  • 非技术背景的办公人员
  • 需要在手机上移动办公
OR

选择 Code 模式

  • 需要编写和调试代码
  • 进行项目架构设计
  • 需要深度定制开发流程
  • 专业的软件开发工作
  • 需要访问本地文件和终端
